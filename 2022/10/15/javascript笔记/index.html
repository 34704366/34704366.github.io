  2. 函数参数的传递：如果是原始类型的值（数值、字符串、布尔值）拷贝按值传递。如果是复合类型的参数，是按地址传递（pass by reference），函数内部修改会影响原始值（形参的修改会影响实参）。

  5. 有时，我们需要在定义函数之后，立即调用该函数。这时，你不能在函数的定义之后加上圆括号，这会产生语法错误。

  6. eval函数没有自己的作用域，有可能修改外部变量的值，造成安全问题，在严格模式中，eval内部声明的变量不会影响到外部作用域，但是仍然可以读写到当前作用域的变量。一般不推荐使用

    • JavaScript 使用一个32位整数，保存数组的元素个数。这意味着，数组成员最多只有 4294967295 个（2^32 - 1）个，也就是说length属性的最大值就是 4294967295。

    • js数组本质上是对象，如果给出超过length属性最大值的数字键而赋值，会自动将数字转为字符串存储。

    • js数组索引实际上是字符串的键来存储。

    • js数组可以存储非数字的键，如果使用for in循环遍历会遍历到非数字键，由于添加非数字键length不会改变，所以for (let i = 0; i < arr.length; i ++) {} 不会遍历到非数字键。

    • js数组中空位和undefined不同，如果某个位置是空位，使用数组的forEach方法、for...in结构、以及Object.keys方法进行遍历，空位都会被跳过。如果显示定义了某个元素是undefined，那么不会被跳过。例如

  8. Number函数

    第一步，调用对象自身的valueOf方法。如果返回原始类型的值，则直接对该值使用Number函数，不再进行后续步骤。

    第二步，如果valueOf方法返回的还是对象，则改为调用对象自身的toString方法。如果toString方法返回原始类型的值，则对该值使用Number函数，不再进行后续步骤。

    第三步，如果toString方法返回的是对象，就报错。

  9. 所有对象（包括空对象）的转换结果都是true，甚至连false对应的布尔对象new Boolean(false)也是true

  11. try…catch…finally语句中，finally语句执行在try/catch的return语句之前，执行在catch语句的throw语句之前。

  12. 使用new object语句创建对象和{}没有区别，后一种方式只是前一种方式的简写。

  13. Objects.getOwnPropertyNames(obj)会返回所有的属性的属性名，Object.keys(obj)，只会返回可遍历的属性名。

  15. Object.preventExtensions 方法可以让一个对象无法再添加新的属性，对应的判断函数是Object.isExtensible

    Object.seal方法使得一个对象既无法添加新属性，也无法删除旧属性，对应的判断函数是Object.isSealed

    Object.freeze方法可以使得一个对象无法添加新属性、无法删除旧属性、也无法改变属性的值，使得这个对象实际上变成了常量。对应的判断函数是Object.isFrozen

  16. Array.map()、forEach() 方法不会跳过undefined和null，但是会跳过空位。

  17. 注意，forEach()方法无法中断执行，总是会将所有成员遍历完。如果希望符合某种条件时，就中断遍历，要使用for循环。

  18. js中字符串是不可变的

  19. 对于空数组，some方法返回false，every方法返回true，回调函数都不会执行。

  20. Array.reduce()第一个参数接收一个累计的起始值，建议总是加上第二个参数，这样比较符合直觉，每个数组成员都会依次执行reduce()方法的参数函数。另外，第二个参数可以防止空数组报错。

  21. indexOf和lastIndexOf两个方法不能用来搜索NaN的位置，即它们无法确定数组成员是否包含NaN。

    这是因为这两个方法内部，使用严格相等运算符（===）进行比较，而NaN是唯一一个不等于自身的值

  22. 会改变原数组的数组方法：

    1. push() / pop()

    2. shift() / unshift()

    3. reverse()

    4. splice()

      splice()作用是删除原来数组中的一部分，并且可以在删除的位置添加新的数组成员

      • 第一个参数是删除的起始位置（从0开始）

      • 第二个参数是被删除的元素个数

      • 如果后面还有更多的参数，则表示这些就是要被插入数组的新元素

    5. sort()， sort函数的参数函数本身接收两个参数，表示进行比较的两个数组成员，如果该参数函数的返回值大于0，表示第一个成员排在第二个成员的后面。

  23. new Number(10) 和 Number(10)的区别：

    new Number(10)是创建一个Number类型的对象（原始类型的包装对象——warpper），直接调用Number(10)是将10转换成数字类型，最后得到的值是原始类型。

  24. "abc".length 如这种方式对字符串调用length属性会自动触发原始类型和实例对象的自动转换。

    abc是一个字符串，本身不是对象，不能调用length属性。JavaScript 引擎自动将其转为包装对象，在这个对象上调用length属性。调用结束后，这个临时对象就会被销毁。这就叫原始类型与实例对象的自动转换。

  25. 空对象、空数组、空函数都会被判断成false

  26. 使用双重的否运算符（!）也可以将任意值转为对应的布尔值。

  27. string的对象方法slice()如果第一个参数大于第二个参数，则返回一个空字符串，subString()方法如果第一个参数大于第二个参数会自动更换两个参数的位置。

    subString()方法不能用负数代表倒数第几个位置，如果参数接收了负数，会自动将其换换成0

  28. 直接调用Date()函数的时候返回的总是当前的时间，即便带有参数

    如果要生成指定参数的时间的话，需要使用new命令

  29. 其他对象求值的时候，都是默认调用.valueOf()方法，但是Date实例求值的时候，默认调用的是toString()方法

  30. Date的set类方法和get类方法，可以结合使用，得到相对时间。

  31. 构造函数中如果return一个非对象的值，return语句会直接忽略它而返回this，没有return语句的情况也是如此，但是注意如果返回一个非this的对象会直接返回该对象。

  32. 判断构造函数是否由new关键字调用：

    1. 判断this的类型 if (! (this instanceof List)) {return new List(/*参数*/)}; 例如在一个名为List的构造器中
    2. 使用函数内部的new.target属性 if (!new.target) {return new List(/*参数*/)};

  33. 数组空元素和undefined不同，数组的forEach方法会跳过空元素，但是不会跳过undefined。

    将数组空元素转换成undefined的方法：

    1. 使用for循环遍历原数组再依次添加到新数组中

    2. 结合apply方法以及Array构造函数

  34. 注意bind函数会返回一个新的函数（新的匿名函数），并非原来函数，这在有些情况会产生问题，比如添加监听事件的时候不能写成这样:element.addEventListener('click', o.m.bind(o));，如果写成这样的话，添加上去的监听函数并非原来的o.m，这就会导致无法使用element.removeEventListener('click', o.m.bind(o)); 移除监听函数。

  35. 对于对象的属性访问的时候，要注意如果是面向对象编程中避免访问对象中不存在的属性，因为会一直查找该对象的原型链直到Object.prototype为止。

  36. h5提出Web Wroker标准，js脚本可以创建多个线程，但是子线程不能操作DOM并且完全受主线程控制，并没有改变js语言单线程的本质。

  37. setInterval()函数是定时循环执行任务，需要注意的是和简单的while + setTimeout不同，setInterval()函数的时间间隔是指的两次任务开始的间隔时间，也就是说不考虑任务执行了多久，如果任务执行的时间超出了指定的时间间隔那么下一次任务就会立即执行。

    并且setTimeout的间隔时间也不是一定的，内部机制是如果到时间还需要等待本轮事件循环（event-loop）的所有任务都执行完毕才会转去执行setTimeout的任务。

    特殊例子：setTimeout(f, 0)不会立即执行，而是等待本事件循环（event-loop）执行完毕再执行（补充：不同浏览器对于0ms的处理策略不同）。

  38. DocumentFragment节点代表一个文档的片段，本身就是一个完整的 DOM 树形结构，可以使用DOM方法向其中添加子节点以及操作，然后可以作为appendChild()、insertBefore()、replaceChild()等方法的参数插入到当前文档，一旦DocumentFragment节点被添加进当前文档，它自身就变成了空节点，通常如此操作比直接操作DOM速度快。

  39. Element.style返回的只是行内样式，并不是该元素的全部样式。通过样式表设置的样式，或者从父元素继承的样式，无法通过这个属性得到。元素的全部样式要通过window.getComputedStyle()得到。

  40. stopPropagation方法只会阻止事件的传播，不会阻止该事件触发当前节点的其他click事件的监听函数。也就是说，不是彻底取消click事件。如果想要彻底取消该事件，不再触发后面所有click的监听函数，可以使用stopImmediatePropagation方法。

  41. 如果使用了beforeunload事件，或者unload事件，浏览器就不会缓存当前网页，因为监听这个事件的目的，一般是为了网页状态，这时缓存页面的初始状态就没意义了。

  42. load事件在页面或者某个资源加载成功的时候触发，但是在页面从缓存中加载时候不触发，如果想要在缓存加载出来页面也触发应该用pageshow事件

  43. 重流与重绘

    • 渲染树转换为网页布局：布局流（flow），布局显示到页面的过程：绘制（paint）。都具有阻塞的效应，消耗时间和计算资源。
    • 重流必然会导致重绘，重绘不一定需要重流。
    • 为了针对重流和重绘进行优化，尽量一次性改变所有css class，不要一项一项的改变。动画使用absolute定位或fixed定位，这样可以减少对其他元素的影响。使用window.requestAnimationFrame()，因为它可以把代码推迟到下一次重绘之前执行，而不是立即要求页面重绘。

  44. 原始类型也可以解构赋值，解构赋值的规则是，只要等号右边的值不是对象或数组，就先将其转为对象。由于undefined和null无法转为对象，所以对它们进行解构赋值，都会报错。

  45. es6新增的三种字符串实例方法：includes(), startsWith(), endsWith()，返回值都是boolean值，第一个参数表示要搜索的内容，第二个参数表示开始搜索的位置（endsWith的行为与其他两个方法有所不同。它针对前n个字符，而其他两个方法针对从第n个位置直到字符串结束。）

  46. js的数组、字符串不支持像python一样负数索引代表倒数第多少个的访问。 字符串方法at()可以支持负数索引。

  47. 字符串的实例函数repeat()如果接收参数是非法的则默认的转换为0，如果<=-1则会报错。

  48. isFinite()和isNaN()的区别在于，传统方法先调用Number()将非数值的值转为数值，再进行判断，而这两个新方法只对数值有效，Number.isFinite()对于非数值一律返回false, Number.isNaN()只有对于NaN才返回true，非NaN一律返回false
